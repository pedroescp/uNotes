# uNotes

Banco de dados: 
 - Usuario
  - Id
  - Nickname
  - Login
  - Senha
  - Email
  - Avatar
  - Telefone
  - CargoId
  - DataInclusao
 - Cargo
  - Id
  - Nome
  - Descricao
 - Grupo
  - Id
  - Nome
 - UsuarioGrupo
  - Id
  - UsuarioId
  - GrupoId
  - DataInclusao
  - DataExclusao
 - Notes
  - Id
  - Titulo
  - Texto
  - CriadorId
  - Lixeira bit
  - Fixado bit
  - DataInclusao
  - DataAtualizacao
 - Colaboradores
  - Id
  - UsuarioId
  - NotaId
  - Status [Pendente, Aceito, Negado]
